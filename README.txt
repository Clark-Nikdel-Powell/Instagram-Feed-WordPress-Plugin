Delivers HTML for displaying a feed of images from Instagram. Includes hooks for outputting additional markup.

== Installation ==

1. Upload the `instagram-feed` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Set the Instagram access token on the Instagram Feed settings page under Settings.
